Prerequisite:
  NodeJS v18.13.0
  NPM v8.19.3

Installation:
  1. Run 'npm install' to install dependencies
  2. Create a new .env file and set the appropriate port in PORT variable and MongoDB connection string in DB_CONNECTION_STRING variable.
  3. Run 'npm start' to start the NodeJS server on Port 3000.


Notes:
  1. All the required features are implemented. 
  2. From Bonus section, point 2 and 4 are implemented.
  3. You can refer to the Postman Collection for API endpoints.
  4. All the endpoints implementations can be found in routers/job.router.js
